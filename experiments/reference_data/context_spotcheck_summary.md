# Context Spot Check (Enron Sample)

- Sample size: **20,000**
- Reply/forward prefix share: **36.9%**
- Attachment-reference share: **18.6%**
- Prior-discussion reference share: **3.5%**
- Internal shorthand/domain acronym share: **30.9%**
- Minimal-subject share: **19.5%**
- Context proxy (>=1 cue): **71.0%**
- Context proxy (>=2 cues): **30.7%**

## Interpretation
- Many messages are not self-contained; they assume thread history, attachments, or org/domain shorthand.
- A non-trivial subset likely requires multi-message memory to classify/answer correctly.
